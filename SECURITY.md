# Reporting Security Vulnerabilities

If you believe you have found a security vulnerability, please **do not** open a
public GitHub issue. Instead, please report it via
[GitHub Security Advisories](https://github.com/shrug-labs/aipack/security/advisories/new).

We ask that you do not use other channels or contact the project maintainers
directly.

Non-vulnerability related security issues including ideas for new or improved
security features are welcome on GitHub Issues.

## Security-Related Information

We will provide security related information such as a threat model, considerations
for secure use, or any known security issues in our documentation. Please note
that labs and sample code are intended to demonstrate a concept and may not be
sufficiently hardened for production use.
